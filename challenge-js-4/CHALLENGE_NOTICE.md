## Challenge Materials

The following files and directories contain additional challenge materials created for this coding challenge:

- `README.md` - Challenge description and instructions
- `challenge.config.json` - Challenge metadata
- `jest.config.cjs` - Jest test configuration for the challenge
- `test/challenge/` - Challenge-specific test files
  - `median.test.js`
